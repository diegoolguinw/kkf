KKF: Kernel Koopman Kalman Filter
==================================

KKF combines Koopman operator theory (via kernel EDMD) with Kalman filtering
for state estimation in nonlinear dynamical systems.

.. code-block:: python

   from kkf import DynamicalSystem, KoopmanOperator, apply_koopman_kalman_filter

Installation
------------

.. code-block:: bash

   pip install kkf
